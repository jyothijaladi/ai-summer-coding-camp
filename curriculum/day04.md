# 💬 Day 4: The Conversation & If/Else

**Time:** 9:00 AM - 12:00 PM
**Daily Goal:** Learn how to get information from the user and make the code "think" (If/Else).

---

## 🕒 9:00 AM – 9:40 AM: Concept 1 – Getting Answers (`input`) 💬
*How to listen to the user.*

1.  **The Input Command:** 
    - `name = input("What is your name? ")`
    - Explain: When Python hits this line, it *waits*. It's listening for the user to type something and hit Enter.
2.  **The "Input is always a String" Rule:**
    - CRITICAL: Even if you type "10", Python thinks it's a string `"10"`. Show them how to turn it into a number with `int(input(...))`.
3.  **The AI Connection:**
    - Ask Gemini: "Why does my code crash when I try to add 5 to `input('Enter a number: ')`?" (Introduce the concept of Type Conversion).

---

## 🕒 9:40 AM – 10:00 AM: 🍎 Brain Break (20 mins)

---

## 🕒 10:00 AM – 10:40 AM: Concept 2 – Making Decisions (If/Else) 🚦
*The logic gate.*

1.  **If/Else Structure:**
    - If `something is True`: do this.
    - Else: do that.
    - *Example:* `if age > 12: print("You can ride!")`
2.  **The Indentation Rule:**
    - Explain that Python uses "Four Spaces" (the Tab key) to know what belongs inside the `if` block.
3.  **The AI Connection:**
    - Task: Ask Gemini: "Write a simple if/else statement that checks if a user's password is 'secret'."

---

## 🕒 10:40 AM – 11:00 AM: 🥤 Mini-Break / Buffer

---

## 🕒 11:00 AM – 12:00 PM: 🏝️ The Sandbox Exploration (Level 4)
*Goal: Build a "Robot Security Guard".*

**Challenges for the Students:**
1.  **The Doorman:** Use `input()` to ask for a password. If it matches your secret word, print "Access Granted." Otherwise, print "Intruder Alert!"
2.  **The VIP Checker:** Ask for the user's age. If they are older than 18, print "Welcome to the VIP lounge."
3.  **The AI Boss Fight:** 
    - Open Antigravity Gemini.
    - Prompt: "I need to ask the user a math question (5 + 5). Write a code snippet that takes their input, converts it to an integer, and tells them if they are right or wrong."

---

### 💡 Teacher's Tip:
*Indentation is the hardest part of Day 4. If a student's code isn't running, it’s almost always because they forgot to indent the code under the `if` or `else` lines. Remind them: "If there's a colon (:), the next line must take a step in!"*
