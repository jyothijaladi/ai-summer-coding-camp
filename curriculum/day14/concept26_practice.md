# 🛠️ Concept 26 Practice: The Dice Roll (Random)

### 🚀 The 20-Problem Challenge Menu
*Teachers: Make sure they `import random` at the top of every file today!*

#### 🟢 Group A: The Number Rollers (Warmups)
1.  **Simple Dice:** Roll a number between 1 and 6 and print it.
2.  **The Decade:** Roll a number between 10 and 20.
3.  **The Coin Flip:** Use `random.choice(["Heads", "Tails"])`.
4.  **Big Winner:** Roll a number between 1 and 1,000,000!
5.  **Multi-Roll:** Use a `for` loop to roll a die 5 times.

#### 🟡 Group B: The Game Customizer
6.  **The Loot Box:** Create a list `["Common", "Rare", "Epik"]`. Print a random choice.
7.  **The Damage Variance:** Pick a random number between 10 and 50 and print `f"You dealt {dmg} damage!"`.
8.  **The NPC Namer:** Pick a random name from a `names` list for a new monster.
9.  **The Color Mix:** Combine two random colors from a `colors` list into an f-string.
10. **The Chance Check:** Roll 1-10. If it's a 10, print "CRITICAL HIT!".

#### 🔴 Group C: The Bug Squasher (Fix the Chaos)
11. **The Missing Pack:** Trying to use `randint()` without `import random`.
12. **The Dot Error:** `random_randint(1, 10)` (Fix the missing dot!).
13. **The List Trap:** `random.choice("Blue", "Red")` (Fix it—needs brackets `[]` for a list!).
14. **The Backward Range:** `random.randint(10, 1)` (Fix the order!).
15. **The Spelling Bee:** `import randome` (Fix the typo!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Story Generator:** Ask Gemini: "Write a short Python script that picks a random Hero, a random Weapon, and a random Villain and prints a battle sentence."
17. **The Emoji Rain:** Ask Gemini: "How do I use `random.choice()` to print a single random emoji from a list of 10?"
18. **The Shuffle Pro:** Ask Gemini: "What does `random.shuffle()` do to a list of cards?"
19. **The Real-World Modeler:** Ask Gemini: "How do games like **Fortnite** use randomness to decide where the 'Battle Bus' flies?"
20. **The Challenge:** Ask Gemini: "Give me a challenge that uses `random.random()` to create a 25% chance of finding a secret door!"

3. Pick a random fortune.
4. Use an f-string to reveal their future: `f"{name}, your destiny is: {fortune}!"`
5. *Bonus:* Print a random "Lucky Number" (1-99) at the bottom.
