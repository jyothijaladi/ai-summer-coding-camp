# 🛠️ Concept 19 Practice: The Key/Value Magic (Dictionaries)

### 🚀 The 20-Problem Challenge Menu
*Teachers: This is the day "Database Management" becomes clear for the kids!*

#### 🟢 Group A: The Dictionary Builders (Warmups)
1.  **Create a Dictionary:** Make a dictionary called `pet` with two keys: `name` and `type`.
2.  **Access Data:** Print the `name` of your pet using its key.
3.  **Add a Value:** Add a new key `age` to your dictionary and set it to 3.
4.  **The Updater:** Change the `type` of your pet to something else.
5.  **Print All Keys:** Use `.keys()` to print every label in your dictionary.

#### 🟡 Group B: The Stat Sheet Manager
6.  **The Super Hero:** Create `hero = {"name": "Marvel", "power": "Flying"}`. Print the power.
7.  **The Inventory:** Create a dictionary of 3 items and their quantities.
8.  **The Price Check:** Create a `menu` with 3 food items and prices. Print the price of "Pizza".
9.  **The High Score:** Create a dictionary with names as keys and scores as values.
10. **The Dictionary Loop:** Use a `for` loop to print every KEY in your dictionary.

#### 🔴 Group C: The Bug Squasher (Fix the Labels)
11. **The Brackets:** `data = [name: Alex]` (Fix the brackets!).
12. **The Key Error:** Trying to access `data["age"]` when there is no "age" key.
13. **Missing Quotes:** `data = {name: "Alex"}` (Fix the missing quotes on the key!).
14. **The Colon Trap:** `data = {"name" - "Alex"}` (Fix the separator!).
15. **Duplicate Keys:** `data = {"a": 1, "a": 2}` (What happens to the first "a"? Fix it!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The RPG Guru:** Ask Gemini: "Give me an example of a nested dictionary for a video game world with 2 regions and their bosses."
17. **The JSON Link:** Ask Gemini: "What is the difference between a Python Dictionary and a JSON file?"
18. **The Secure Lookup:** Ask Gemini: "How do I use `.get()` to find a value in a dictionary without crashing the program if the key is missing?"
19. **The Real-World Modeler:** Ask Gemini: "Write a dictionary that stores all the settings for a 'Dark Mode' app (Example: `bg_color`, `text_color`, `font_size`)."
20. **The Challenge:** Ask Gemini: "Give me a hard Python dictionary puzzle involving looping through keys and values at the same time!"

### 🌟 Stretch Goal for "Flash" Students
Build a **"Translating Mirror"**! 
1. Create a dictionary that maps English words to "Emoji" versions (Example: `{"happy": "😊"}`).
2. Ask the user to input a word.
3. If the word is in the dictionary, print the emoji.
4. If not, print "Word not found in the Mirror!"
