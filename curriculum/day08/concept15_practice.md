# 🛠️ Concept 15 Practice: Processing Lists (Iteration)

### 🚀 The 20-Problem Challenge Menu
*Note: This is how we handle THOUSANDS of items at once!*

#### 🟢 Group A: The Basic Scan (Warmups)
1.  **Print a List:** Create a list of 5 animals. Use a loop to print each one.
2.  **Number Shout:** Create a list of 5 numbers. Print each one multiplied by 10.
3.  **The Greeter:** Create a list of 3 names. Print "Hello, [Name]!" for each.
4.  **Length Check:** For each word in a list, print how many letters it has.
5.  **Shouty List:** For each word in a list, print it in ALL CAPS.

#### 🟡 Group B: The Inventory Logic
6.  **Find the Sword:** In a list of items, use a loop and an `if` to print "FOUND IT!" when you see "Sword".
7.  **Count the Evens:** In a list of numbers, count how many are even (use `% 2 == 0`).
8.  **The Totalizer:** Create a list of 5 prices. Add them all up and print the final total.
9.  **The Filter:** From a list of names, only print the ones that start with the letter "A".
10. **The Update:** Create a list of hero levels. For each level, add 1 and print the new level.

#### 🔴 Group C: The Bug Squasher (Fix the Iterator)
11. **The Plural Mistake:** `for friend in friends: print(friends)` (Fix it—you should print `friend`!).
12. **The Logic Gap:** A loop that is supposed to find a specific item but has no `if` check.
13. **Bracket Confusion:** `for i in [1, 2, 3]` (Is the list inside the loop? Fix the missing colon!).
14. **The Integer Trap:** Trying to use `.upper()` on a list of numbers.
15. **The Ghost Variable:** `for x in my_list: print(y)` (Fix it—the names don't match!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Movie Recommender:** Ask Gemini: "Give me a list of 5 movie titles and a `for` loop that prints 'I highly recommend [Title]' for each."
17. **The Color Changer:** Ask Gemini: "How do I use a `for` loop to change all the strings in a list into lower-case?"
18. **The Data Picker:** Ask Gemini: "How do I only print the numbers GREATER than 50 from a list of 10 random numbers?"
19. **The Real-World Modeler:** Ask Gemini: "Write a script that calculates the total cost of a shopping cart list including a 10% tax for each item."
20. **The Challenge:** Ask Gemini: "What is an 'f-string' within a `for` loop? Show me a cool example using a guest list."
