# 🛠️ Concept 14 Practice: The Repeat Machine (For Loops)

### 🚀 The 20-Problem Challenge Menu
*Teachers: This is where kids realize they can do 1,000 tasks with 2 lines of code!*

#### 🟢 Group A: The Counting Warmups
1.  **Count to 10:** Print numbers 0 to 9 using `range(10)`.
2.  **Start at 1:** Print numbers 1 to 10 using `range(1, 11)`.
3.  **The Message:** Print "I love coding!" 5 times in a row.
4.  **Count by Twos:** Print even numbers from 2 to 20 using `range(2, 21, 2)`.
5.  **The Countdown:** Print 10 down to 1 using `range(10, 0, -1)`.

#### 🟡 Group B: Real World Repetition
6.  **The Monster Spawner:** Print "A new Zombie has appeared!" 3 times.
7.  **The Squat Challenge:** Print "Squat #1", "Squat #2", etc., up to 10.
8.  **The Loading Bar:** Use a loop to print 50 dots (`.`) in a row.
9.  **The Multiplier:** Use a loop to print the "5 Times Table" (5, 10, 15... up to 50).
10. **The Year Tracker:** Print every year from 2000 to 2024.

#### 🔴 Group C: The Bug Squasher (Fix the Loops)
11. **The Missing Colon:** `for i in range(5) print(i)` (Fix it!).
12. **The Indent Error:** 
    ```python
    for i in range(3):
    print("Hi") # Fix the alignment!
    ```
13. **The Variable Mistake:** `for num in range(5): print(i)` (Fix it—the variable names don't match!).
14. **The Endless Loop:** A logic error where the loop doesn't have a range (HINT: Check if `range` is spelled right!).
15. **The Zero Trap:** A loop meant to run 10 times but only running 9.

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The ASCII Wave:** Ask Gemini: "Write a `for` loop that prints a pattern of symbols that looks like a wave."
17. **The Emoji Rain:** Ask Gemini: "Write a loop that prints 10 random rain emojis."
18. **The Math Master:** Ask Gemini: "How do I use a `for` loop to add up all the numbers from 1 to 100?"
19. **The Real-World Modeler:** Ask Gemini: "How does a website like Amazon use a `for` loop to show all the products in my shopping cart?"
20. **The Challenge:** Ask Gemini: "Give me a hard Python loop challenge that involves the 'Modulo' operator!"

### 🌟 Stretch Goal for "Flash" Students
Can you build a **"Rocket Launcher"**? 
1. Ask the user for a countdown number (Example: 20).
2. Use a `for` loop with `range` to count down to 0.
3. At the end, print a giant "BLAST OFF! 🚀" using ASCII art.
4. *Bonus:* For every second, print a "Fuel Level" that goes down!
