# 🛠️ Concept 9 Practice: The Smart Switches

### 🚀 The 20-Problem Challenge Menu
*Teachers: This is the shortest code they’ll write, but the most important!*

#### 🟢 Group A: The Toggle Warmups
1.  **The Light Switch:** Create `is_on = True`. Print it.
2.  **The Opposite:** Create `is_sad = False`. Print it.
3.  **The Flip:** Print the result of `not True`.
4.  **The Double Flip:** Print the result of `not not False`.
5.  **Multi-Toggle:** Create 3 Booleans for `hp_low`, `mana_low`, and `is_poisoned`. Set them all to `False`.

#### 🟡 Group B: Comparisons to Booleans (The Secret Link)
6.  **The Math Check:** Print `5 > 3`. (Observe the output!)
7.  **The Equal Check:** Print `10 == 10`.
8.  **The Name Check:** Create `name = "Alex"`. Print `name == "Alex"`.
9.  **The Mystery Number:** Create `num = 7`. Print `num < 0`.
10. **The Gap Check:** Print `100 != 50`.

#### 🔴 Group C: The Bug Squasher (Fix the Switches)
11. **Small Letter:** `is_ready = true` (Fix it!).
12. **The String Trap:** `is_running = "True"` (Fix it by removing the quotes!).
13. **Missing Value:** `is_online = ` (Finish it!).
14. **Reserved Word:** `True = "Yes"` (Fix it—you can’t use True as a variable name!).
15. **The Logic Loop:** `is_open = not "Open"` (Fix the logic error!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Game Architect:** Ask Gemini: "Give me a list of 5 Boolean variables an 'Among Us' server would use to track a player."
17. **The Truth Finder:** Ask Gemini: "What is the 'Truth Table' for the `not` operator in Python?"
18. **The Hardware Pro:** Ask Gemini: "How do Booleans related to the 0s and 1s (Binary) inside a computer chip?"
19. **The Real-World Modeler:** Ask Gemini: "Write an `if` statement that checks if `is_logged_in` is True before showing 'Welcome'."
20. **The Challenge:** Ask Gemini: "Give me a tricky Python logic puzzle where I have to use `not` twice!"

### 🌟 Stretch Goal for "Flash" Students
Can you build a **"Logic Inverter"**? 
1. Ask the user for their name.
2. Create a variable `is_cool = True`.
3. If they enter a "Forbidden" name (like "Lord Voldemort"), set `is_cool` to `False`.
4. Use an f-string to tell them their status: `f"Status of {name}: {is_cool}"`!
