# 🛠️ Concept 12 Practice: The Backpack (Lists)

### 🚀 The 20-Problem Challenge Menu
*Teachers: This is the day "Inventory Management" becomes real for the kids!*

#### 🟢 Group A: The List Builders (Warmups)
1.  **Create a List:** Make a list called `fruits` with 3 items.
2.  **The First Item:** Print the item at index `0`.
3.  **The Final Item:** Print the last item in a list of 5 things.
4.  **Count Them:** Use `len()` to print how many items are in your list.
5.  **Multi-Print:** Create a list and print the whole thing at once.

#### 🟡 Group B: The Inventory Manager (Adding & Removing)
6.  **Add an Item:** Create `inventory = ["Sword"]`. Use `.append("Shield")` and print it.
7.  **The New Hero:** Add a new friend's name to the end of a `friends` list.
8.  **Remove an Item:** Use `.pop()` to remove the last item from a list.
9.  **Clear the Trash:** Use `.remove()` to delete a specific item by name.
10. **The Update:** Change the item at index `1` to something else (Example: `inv[1] = "Fire Bow"`).

#### 🔴 Group C: The Bug Squasher (Fix the Shelves)
11. **The Brackets:** `fruits = (Apple, Banana)` (Fix the brackets and missing quotes!).
12. **The Zero Trap:** Trying to print `my_list[3]` when the list only has 3 items. (HINT: The max index is 2!).
13. **Missing Commas:** `list = ["Red" "Blue"]` (Fix the missing comma!).
14. **Method Confusion:** `list.add("Green")` (Fix it—Python uses `.append()`, not `.add()`).
15. **The Capital Trap:** `List = []` (Fix the variable name—avoid using reserved words like 'List' with a capital!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The RPG Guru:** Ask Gemini: "Give me a Python list of 5 'Rare Items' for a pirate game."
17. **The Slicer:** Ask Gemini: "How do I print only the middle 3 items of a long list?"
18. **The Sorter:** Ask Gemini: "How do I sort my list alphabetically in one line of code?"
19. **The Real-World Modeler:** Ask Gemini: "How does a Netflix 'Continue Watching' list work in Python? Write a small example."
20. **The Challenge:** Ask Gemini: "What is the difference between `list.pop()` and `list.remove()`? When should I use each one?"

### 🌟 Stretch Goal for "Flash" Students
Build a **"Party Guest List"** script! 
1. Create an empty list called `guests`.
2. Ask the user for 3 names using `input()` and `.append()` them.
3. Remove the 2nd guest (they couldn't make it!).
4. Add "Antigravity AI" to the end of the list.
5. Print the final list with an f-string: `f"The party starts with {guests}!"`
