# 🛠️ Concept 13 Practice: The Frozen Vault (Tuples)

### 🚀 The 20-Problem Challenge Menu
*Teachers: This is about understanding "Reading" vs "Writing" data!*

#### 🟢 Group A: The Tuple Builders (Warmups)
1.  **Create a Tuple:** Make a tuple called `stats` with 3 numbers.
2.  **Access Data:** Print the 2nd item in your tuple (Notice the index!).
3.  **The Proof:** Try to change an item in your tuple. Watch the computer complain!
4.  **The Count:** Use `len()` to see how many items are in your tuple.
5.  **Multiplication:** Print a tuple 3 times in a row (Example: `("Win",) * 3`).

#### 🟡 Group B: Real World Coordinates
6.  **The X/Y Point:** Create a tuple `player_pos = (50, 75)`. Print the X and Y separately.
7.  **The Birthday Vault:** Store your birth year, month, and day in a tuple.
8.  **The Color Code:** Store an RGB color `(0, 255, 0)` and print the Green value.
9.  **The Box Size:** Create `dimensions = (1920, 1080)`.
10. **The Unpacking:** Create `point = (1, 2)`. Use `x, y = point` to separate them!

#### 🔴 Group C: The Bug Squasher (Fix the Vault)
11. **The Forbidden Change:** `tup = (1, 2)` then `tup[0] = 5` (Explain why this fails!).
12. **The Missing Comma:** `single = ("Secret")` (Is this a tuple? Check the type!).
13. **Method Madness:** `tup.append(3)` (Fix it—you can’t append to a tuple!).
14. **The Bracket Mix:** `tup = [1, 2)` (Fix the mismatched brackets!).
15. **The Index Jump:** Trying to find `tup[5]` when the tuple only has 2 items.

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The GPS Guru:** Ask Gemini: "What are the GPS coordinates of the Eiffel Tower? Write them as a Python tuple."
17. **The Speed Champion:** Ask Gemini: "Why are Tuples faster for the computer to process than Lists?"
18. **The Conversion Pro:** Ask Gemini: "How do I turn a List into a Tuple in Python?"
19. **The Real-World Modeler:** Ask Gemini: "Give me an example of how a bank uses Tuples to store a customer's Social Security Number."
20. **The Challenge:** Ask Gemini: "What is an 'Immutable' object and why is a Tuple better than a List for storing the colors of the rainbow?"

### 🌟 Stretch Goal for "Flash" Students
Can you create a **"Safe Search"**?
1. Create a Tuple of "Forbidden Words."
2. Ask the user to input a word.
3. Use `if word in forbidden_tuple:` to check if their word is locked in the vault.
4. Print "Access Denied" if it is!
