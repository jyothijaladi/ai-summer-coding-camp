# 🛠️ Concept 24 Practice: The Output Slot (Returns)

### 🚀 The 20-Problem Challenge Menu
*Note: Today, we "Catch" the answers the computer gives us!*

#### 🟢 Group A: The Math Backwards (Warmups)
1.  **Add Return:** Write `add(a, b)` that RETURNS the sum. Catch it in a variable `x` and print `x`.
2.  **Double Return:** Write `double(n)` that returns `n * 2`.
3.  **Name Fixer:** Write `fix(n)` that returns a name in `.upper()`.
4.  **Is Ten?** Write `is_ten(n)` that returns `True` if `n == 10`.
5.  **Multi-Math:** Write a function that returns `(a + b) * c`.

#### 🟡 Group B: The Game Logic Return
6.  **The Damage Calc:** Define `calc_hit(lvl)`. Return `lvl * 5`.
7.  **The Loot Picker:** Define `get_loot()`. Return a random string from a list.
8.  **The HP Check:** Return `True` if `hp > 0`, otherwise return `False`.
9.  **The Item Finder:** Return `True` if an item is in a global `inventory` list.
10. **The Speed Boost:** Return a new speed value that is `old_speed + 10`.

#### 🔴 Group C: The Bug Squasher (Catch the Answer)
11. **The Ghost Return:** Calling `x = my_func()` when the function only has `print` (What is inside `x`? Hint: It's `None`!).
12. **The Unreachable Code:** Writing code *after* a `return` line inside a function.
13. **Missing Catch:** Calling `calculate(10, 5)` but not saving the answer to a variable.
14. **The Type Mixup:** Returning a number but trying to use it like a string.
15. **Double Return Trap:** Having two `return` lines in a row (Which one runs?).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Area Expert:** Ask Gemini: "Write a function `area(radius)` that calculates the area of a circle and returns it."
17. **The Emoji Switcher:** Ask Gemini: "Write a function that takes a word and returns a matching emoji string."
18. **The Logic Pro:** Ask Gemini: "What is `None` in Python and why do I see it when I print a function call?"
19. **The Real-World Modeler:** Ask Gemini: "Write a function for a shopping cart that returns the total price after a 10% discount."
20. **The Challenge:** Ask Gemini: "Give me a challenge where I have to use the output of one function as the input for a second function!"

### 🌟 Stretch Goal for "Flash" Students
Can you build a **"Price Scanner"**? 
1. Create a `prices` dictionary.
2. Define `get_total(item1, item2)`.
3. Inside the function, look up both prices, add them, and **RETURN** the total.
4. Catch the total and add a "Tax" of 5% in your main code!
