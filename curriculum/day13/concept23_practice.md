# 🛠️ Concept 23 Practice: The Input Slot (Parameters)

### 🚀 The 20-Problem Challenge Menu
*Note: Make sure your "Plugs" match your "Slots"!*

#### 🟢 Group A: The Personalized Machines (Warmups)
1.  **Name Machine:** Define `hi(name)`. Call it with your name.
2.  **Double Machine:** Define `double(num)`. Print the number times 2.
3.  **Color Splash:** Define `paint(color)`. Print `f"Painting {color}!"`.
4.  **The Squarer:** Define `square(n)`. Print `n * n`.
5.  **Multi-Hello:** Define `greet(name, city)`. Print `f"Hi {name} from {city}!"`.

#### 🟡 Group B: The Game Customizer
6.  **Damage Bot:** Define `hit(enemy, damage)`. Print `f"You hit {enemy} for {damage}!"`.
7.  **Inventory Adder:** Define `pickup(item)`. Add the item to a global list.
8.  **The Score Booster:** Define `win(points)`. Add the points to a `total_score`.
9.  **The Level Switch:** Define `jump_to(level_num)`. Print `f"Teleporting to Level {level_num}..."`.
10. **The Speedster:** Define `vroom(speed)`. If speed is over 100, print "FAST!".

#### 🔴 Group C: The Bug Squasher (Fix the Plugs)
11. **The Missing Plugin:** Calling `greet()` when it needs `greet(name)`.
12. **The Extra Plugin:** Calling `hi("Alex", "Extra")` when it only needs one name.
13. **The Shadow Variable:** Trying to use the parameter name outside the function.
14. **The Type Trap:** Sending a string `"10"` to a function that tries to do math.
15. **The Colon Mistake:** `def hi(name) print(name)` (Fix it!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Storyteller:** Ask Gemini: "Write a function called `tell_story(hero, animal)` that uses both parameters in a funny sentence."
17. **The Color Mixer:** Ask Gemini: "Write a function `mix(c1, c2)` that prints 'Mixing [c1] and [c2] to make something new!'."
18. **The Math Master:** Ask Gemini: "How do I make a parameter optional so I don't HAVE to type it every time?"
19. **The Real-World Modeler:** Ask Gemini: "Write a function for an Uber app called `request_ride(pickup, destination)`."
20. **The Challenge:** Ask Gemini: "Give me a challenge that uses 3 different parameters of different types (String, Number, and Boolean)!"

### 🌟 Stretch Goal for "Flash" Students
Can you build a **"Super Calculator"**? 
1. Define 4 functions: `add(a, b)`, `sub(a, b)`, `mult(a, b)`, and `div(a, b)`.
2. Each function should print the result.
3. Ask the user for two numbers and an operation (+, -, *, /).
4. Use an `if` statement to call the correct function with the two numbers!
