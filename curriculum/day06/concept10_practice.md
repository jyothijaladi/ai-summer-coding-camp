# 🛠️ Concept 10 Practice: The Logic Gates

### 🚀 The 20-Problem Challenge Menu
*Note: Python evaluates `and` first, then `or`. Be careful!*

#### 🟢 Group A: Simple Boolean Math (Warmups)
1.  **Double True:** Print `True and True`.
2.  **The Mix:** Print `True and False`.
3.  **The Lifeline:** Print `True or False`.
4.  **The Failure:** Print `False and False`.
5.  **Triple Threat:** Print `True or False or False`.

#### 🟡 Group B: Real World Comparisons
6.  **The Age Gate:** Check if `age > 10` AND `age < 15`.
7.  **The Freebie:** Check if `is_birthday == True` OR `is_holiday == True`.
8.  **The Secure Entry:** Check if `has_id == True` AND `knows_password == True`.
9.  **The Weather Prep:** Check if `is_raining == True` OR `is_snowing == True`.
10. **The Game Over:** Check if `lives == 0` AND `has_continue == False`.

#### 🔴 Group C: The Bug Squasher (Fix the Logic)
11. **Text vs Logic:** `if x == 5 and 10:` (Fix it—you need `x == 10` on the second part too!).
12. **The Double Check:** `if has_key and level = 5:` (Fix the assignment vs comparison error!).
13. **The Logical Flip:** `if not is_online or True` (Is this always True? Fix the logic if it's redundant!).
14. **Missing Parentheses:** `if (5 > 3 or 2 < 1) and 10 == 10:` (Verify the order of operations!).
15. **The Spelling Bee:** `if x is 5 AND y is 10:` (Fix the capital "AND"—Python uses lowercase!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Game Security:** Ask Gemini: "Write a Python check that makes sure a player has enough 'Energy' AND 'Coins' to buy a spaceship."
17. **The Truth Table:** Ask Gemini: "Can you draw me a Truth Table for the `or` operator?"
18. **The Short-Circuit:** Ask Gemini: "What is 'Short-Circuit Evaluation' in Python logic?"
19. **The Real-World Modeler:** Ask Gemini: "How does a Netflix 'Kid Profile' use logic to only show PG-rated movies?"
20. **The Challenge:** Ask Gemini: "Give me a logic puzzle that uses `and`, `or`, and `not` all in one line of code!"
