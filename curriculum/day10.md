## 🕒 9:00 AM – 9:40 AM: Concept – Loops Inside Loops (Nested) 🌀
*Creating grids and complex patterns.*

1.  **The "Nested" Concept:** 
    - Imagine a clock: The hour hand only moves once every time the minute hand goes around 60 times. That's a nested loop!
2.  **Making a Grid:**
    - Use two `for` loops to print a row and column of stars `*`.
    - `for x in range(3): for y in range(3): print("*")`
3.  **The AI Connection:**
    - Ask Gemini: "Show me a nested loop in Python that prints a 5x5 grid of emojis."

## 🕒 10:00 AM – 12:00 PM: 🏆 FRIDAY HERO PROJECT: The Virtual AI Pet 🐱
*Goal: Create a "Tamagotchi" style pet that tracks stats and reacts to user input in a constant loop.*

### The Project Blueprint:
1.  **The Pet Setup:** Create variables for `pet_name`, `hunger` (50), and `happiness` (50).
2.  **The Game Loop:** Use a `while True` loop to keep the game running until the user types "exit".
3.  **The Action Menu:** Inside the loop, ask the user what they want to do: `1. Feed`, `2. Play`, `3. Status`, `4. Exit`.
4.  **The Logic:** Use `if/elif/else` statements to change the hunger/happiness stats based on the user's choice.
5.  **The Secret Event:** If `hunger` gets too high (above 80) or `happiness` too low (below 20), make the pet "run away" and break the loop!

- **Adding Features:** Ask Gemini: "How do I add a 'Sleep' action to my pet game that resets the hunger stat?"
