# 🛠️ Concept 32 Practice: The AI Whisperer (Prompting)

### 🚀 The 20-Problem Challenge Menu
*Teachers: This is a lab day! Students should use Antigravity Gemini for every task!*

#### 🟢 Group A: The AI Starters (Warmups)
1.  **The Greeter:** Ask the AI to write a function that greets you in 5 different languages.
2.  **The Loop Maker:** Ask for a 3-line `for` loop that prints every even number between 1 and 20.
3.  **The Emoji Code:** Ask the AI to give you a list of 10 "Space" emojis.
4.  **The Secret Finder:** Ask: "What is the difference between a List and a Tuple in one sentence?"
5.  **The Code Cleaner:** Paste a piece of messy code and ask: "Can you make this code look professional?"

#### 🟡 Group B: The Creative Partner
6.  **The Game Idea:** Ask Gemini for 5 ideas for a game about a "Coding Cat."
7.  **The Story Mix:** Ask: "Write a 5-sentence story that includes a `loop`, a `variable`, and a `giant robot`."
8.  **The ASCII Pro:** Ask for an ASCII art of a dragon.
9.  **The Variable Namer:** Ask: "Give me 10 cool variable names for a 'Time Traveler' character."
10. **The Dialogue Bot:** Ask for 3 funny lines an AI villain would say when they win.

#### 🔴 Group C: The Bug Hunter (Troubleshooting)
11. **The Infinite Loop:** Write an infinite loop and ask Gemini: "Why does this never stop and how do I fix it?"
12. **The Missing Colon:** Write code with a missing colon and ask: "What is wrong with this line?"
13. **The Logic Error:** Write code that subtracts when it should add. Ask: "Is there a math mistake here?"
14. **The Indent Issue:** Ask Gemini to "Fix the indentation of this block of code."
15. **The Type Trap:** Ask: "Why can't I add a string and an integer together in Python?"

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Challenge Creator:** Ask Gemini: "Give me a hard Python challenge for a Day 19 student!"
17. **The Explainer:** Ask Gemini: "How does a 'Search Algorithm' work using a 'Supermarket' analogy?"
18. **The Mock Interview:** Ask Gemini: "Act like a lead developer at Google. Ask me 3 questions to see if I know Python."
19. **The Real-World Modeler:** Ask Gemini: "How do big companies like **Netflix** use AI to decide which movies I might like?"
20. **The Master Prompt:** Ask Gemini: "What is the best way to write a prompt so you give me the most accurate code possible?"

---

### 🌟 Stretch Goal for "Flash" Students
Can you build an **"AI Secret Message Decoder"**? 
1. Ask Gemini to write a "Secret Message" in a "Rot13" or "Caesar Cipher" code.
2. Write a Python script (with Gemini's help!) that can read that secret message and "Decode" it back into English.
3. Save the secret message to a file, then load it and decode it!
