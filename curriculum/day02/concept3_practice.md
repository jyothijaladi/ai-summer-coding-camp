# 🛠️ Concept 1 Practice: The Calculator Lab

### 🚀 The 20-Problem Challenge Menu
*Teachers: Let students explore the "Modulo" (%) challenges for a real brain-workout!*

#### 🟢 Group A: The Basic Operation Warmups
1.  **Add it Up:** Print the result of `15 + 25`.
2.  **Take it Away:** Create a variable `apples = 100`, subtract 20, and print the result.
3.  **Multiply Mastery:** Print the result of `5 * 5`.
4.  **The Great Divide:** Divide `100` by `10` and print the result.
5.  **Triple Combo:** Print the result of `(10 + 5) * 2`.

#### 🟡 Group B: The Word Problem Challenge
6.  **The Pizza Party:** You have 3 pizzas with 8 slices each. Print the total slices.
7.  **The Allowance:** You get $5 a day. Print how much you have after 7 days.
8.  **The Shared Loot:** You found 50 gold coins and have 5 teammates. Print how many each person gets.
9.  **The Weight Lifter:** You have two 10kg weights and one 20kg weight. Print the total weight.
10. **The Speedster:** You ran 10 miles in 2 hours. Print your speed (miles divided by hours).

#### 🔴 Group C: The Brain Teasers (Logic & Remainder)
11. **The Remainder:** Use the `%` operator to find the remainder of `21 / 4`.
12. **Even or Odd?** Print `10 % 2` and `11 % 2`. What’s the difference?
13. **The Float Fix:** Divide `7` by `2`. Is the answer a whole number or a decimal?
14. **The Big Reset:** Multiply `1234` by `0`. What happens?
15. **The Power of Order:** Print `10 + 2 * 5`. Now print `(10 + 2) * 5`. Why are they different?

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Game Designer:** Ask Gemini: "I want to create a damage system. Write a Python line where player health is subtracted by a random number between 1 and 10."
17. **The Scientist:** Ask Gemini: "How do I convert Celsius to Fahrenheit using Python math?"
18. **The Money Pro:** Ask Gemini: "Write a Python script that calculates a 15% tip for a $40 dinner bill."
19. **The Time Traveler:** Ask Gemini: "How many seconds are in a 30-day month? Write a Python calculation for it."
20. **The Mystery Op:** Ask Gemini: "What is the `**` operator in Python and what does it do to numbers?"

### 🌟 Stretch Goal for "Flash" Students
Can you create a **Shopping Receipt**? 
1. Create 3 variables for item prices.
2. Calculate the subtotal.
3. Add a 5% tax.
4. Print the final total with a "Thank You" message!
