# 🛠️ Concept 2 Practice: Smart Numbers (Comparisons)

### 🚀 The 20-Problem Challenge Menu
*Note: Every time you run a comparison, Python will tell you: `True` or `False`.*

#### 🟢 Group A: The Comparison Warmups
1.  **Greater than:** Print the result of `10 > 5`.
2.  **Less than:** Print the result of `100 < 10`.
3.  **The Equal Check:** Create `a = 5` and `b = 5`. Print `a == b`.
4.  **The Not Equal:** Create `c = 10` and `d = 20`. Print `c != d`.
5.  **Greater or Equal:** Print the result of `10 >= 10`.

#### 🟡 Group B: The Game Logic Puzzles
6.  **The Level Lock:** Create a variable `player_level = 8`. Print if `player_level` is greater than or equal to 10.
7.  **Out of Ammo:** Create `ammo = 0`. Print if `ammo` is equal to 0.
8.  **The Winner Circle:** Create `my_score = 500` and `high_score = 450`. Print if you beat the record!
9.  **The Inventory Full:** Create `items = 12`. Print if `items` is less than or equal to 10.
10. **Correct Pin:** Create `correct_pin = 1234` and `user_entry = 4321`. Print if they match.

#### 🔴 Group C: The Brain Teasers
11. **Negative Numbers:** Print if `-5 < 0`.
12. **The Float Mix:** Print if `5.0 == 5`. Does Python think these are the same?
13. **The Math Comparison:** Print if `(5 + 5) == (2 * 5)`.
14. **The Zero Check:** Create `health = 100`. Print if `health != 0`.
15. **Chain of Logic:** Create `x = 5`. Print if `3 < x < 10`.

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Movie Guard:** Ask Gemini: "Write a Python comparison that checks if a user is older than 13 OR exactly 13 to watch a movie."
17. **The Temperature Monitor:** Ask Gemini: "Write a Python check that warns if a variable `fridge_temp` is greater than 40 degrees."
18. **The Basketball Scout:** Ask Gemini: "Write a Python comparison that checks if a `player_height` is greater than 6 feet."
19. **The Password Pro:** Ask Gemini: "How do I compare two strings to see if a password is correct?"
20. **The Challenge:** Ask Gemini: "Give me a tricky Python math comparison where the answer is 'False'."

2. Use comparisons to determine if it's a "Draw" or if one player is "Leading."
