# 📖 Day 11: Dictionaries & The Unique Set

**Time:** 9:00 AM - 12:00 PM
**Daily Goal:** Learn how to organize data using Key-Value pairs and how to keep a collection of unique items.

---

## 🕒 9:00 AM – 9:40 AM: Concept 1 – Dictionaries (Labeling your Data) 📖
*Data isn't just a list; it’s a collection of labeled info.*

1.  **Key-Value Pairs:** 
    - `player = {"name": "Zelda", "level": 10, "is_alive": True}`
    - Explain that the "Key" is the label (name) and the "Value" is what's inside (Zelda).
2.  **Updating Info:**
    - `player["level"] = 11` (Congratulations, you leveled up!).
3.  **The AI Connection:**
    - Ask Gemini: "What is a real-world example of a dictionary in programming? (Hint: Think about a profile page or a shopping cart)."

---

## 🕒 9:40 AM – 10:00 AM: 🍎 Brain Break (20 mins)

---

## 🕒 10:00 AM – 10:40 AM: Concept 2 – Sets (The Unique Collectors) 💎
*A collection where no two things can be the same.*

1.  **Working with Sets:**
    - `bag = {"Apple", "Banana", "Apple"}` -> Result: `{"Apple", "Banana"}`.
    - Explain that sets automatically delete duplicates.
2.  **The "Unique" Use Case:**
    - Use sets to keep track of "Badges Earned" or "Unique Monsters Found."
3.  **The AI Connection:**
    - Task: Ask Gemini: "I have a list with double names: `['Alex', 'Bob', 'Alex']`. Write a one-line Python trick to turn it into a Set to remove the duplicates."

## 🕒 11:00 AM – 12:00 PM: 🏝️ The Sandbox Exploration
*Goal: Build a "Student Profile Maker".*

**Challenges for the Students:**
1.  **The ID Card:** Create a dictionary for yourself with 4 keys: `name`, `age`, `favorite_subject`, and `has_pet` (True/False).
2.  **The Sticker Book:** Create an empty set named `stickers`. Ask the user for 3 stickers they found today. Add them to the set. What happens if they type the same sticker twice?
3.  **The AI Boss Fight:** 
    - Open Antigravity Gemini.
    - Prompt: "I have a dictionary `car = {'brand': 'Ford', 'model': 'Mustang'}`. Write a line of code that adds a new key called 'year' set to 2024."
