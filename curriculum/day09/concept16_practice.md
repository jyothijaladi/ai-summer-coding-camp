# 🛠️ Concept 16 Practice: The Logic Loop (While)

### 🚀 The 20-Problem Challenge Menu
*Caution: Make sure your loops have a way to STOP, or you'll be stuck forever!*

#### 🟢 Group A: The Counting Loops (Warmups)
1.  **Count to 10:** Use a `while` loop to print numbers 1 to 10.
2.  **Countdown:** Use a `while` loop to print 10 down to 1.
3.  **The Double:** Start a variable at 1. `while` it is under 100, double it and print.
4.  **Stay High:** `while x < 50: x += 5; print(x)`.
5.  **Multi-Message:** Print "Loading..." 5 times using a `while` loop and a counter.

#### 🟡 Group B: Real World Conditions
6.  **The Password Guard:** Ask for a password `while` the input is not exactly "Secret123".
7.  **The Hungry Panda:** Set `hunger = 100`. `while hunger > 0`, print "Eating..." and subtract 20.
8.  **The Gas Tank:** Start `gas = 10`. `while gas > 0`, print "Driving..." and subtract 1.
9.  **The Coin Collector:** `while coins < 10`, ask the user "Did you find a coin?" and add 1 if they say yes.
10. **The Door Knock:** `while knocks < 3`, print "Knock!" and add 1 to `knocks`.

#### 🔴 Group C: The Bug Squasher (Fix the Infinite Loop)
11. **The Standing Counter:** `i = 0` then `while i < 5: print(i)` (Fix it—needs `i += 1`!).
12. **The Wrong Way:** `i = 10` then `while i > 0: i += 1` (Fix it—the number is going the wrong way!).
13. **The Variable Mistake:** `while x < 5:` when `x` was never created.
14. **The Comparison Swap:** `while x = 10:` (Fix the assignment vs comparison error!).
15. **The Spelling Error:** `whille i < 5:` (Fix the typo!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Menu Maker:** Ask Gemini: "How do I write a `while` loop that keeps asking 'What do you want to do?' until I type 'Home'?"
17. **The Health Tracker:** Ask Gemini: "Write a `while` loop that simulates a player losing 5 health every second until they hit 0."
18. **The Input Guard:** Ask Gemini: "How do I make a `while` loop that only accepts a NUMBER and not a string?"
19. **The Real-World Modeler:** Ask Gemini: "How does a website like Google use a `while` loop to wait for a user to type before showing search results?"
20. **The Challenge:** Ask Gemini: "Give me a `while` loop puzzle where the loop only runs IF the number is prime!"

### 🌟 Stretch Goal for "Flash" Students
Can you build a **"Higher or Lower"** game? 
1. Create a `secret_number = 42`.
2. Use a `while` loop to keep asking the user for a guess.
3. **If** the guess is correct, print "You Win!" and stop the loop.
4. **If** incorrect, tell them to try again!
