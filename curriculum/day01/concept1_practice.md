# 🛠️ Concept 1 Practice: Making Python Talk

### 🏁 Your First Command
Type this into your Python editor and hit Run:

```python
print("Hello World!")
```

### 🧪 Experiment Time
Now, try to change the message! Can you make the computer say:
- Your name?
- Your favorite food?
- A funny sound?

---

### 🤖 The AI Prompt Challenge
Open **Antigravity Gemini** and type this prompt:

> "I just ran the `print('Hello World')` command. Can you explain why I need to use quotation marks around the message? What happens if I forget them?"

---

### 💡 Hint
If the computer gives you a red error message, check your **Quotation Marks**! Both sides of the message must have one.
