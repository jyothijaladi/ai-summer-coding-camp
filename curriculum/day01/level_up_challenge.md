# 🏆 Day 1 Level Up Challenge: The Stat Board 🚀

**Goal:** Use everything you learned today (Printing and Variables) to build a character profile for your favorite Game or Movie character.

---

### 📋 The Mission
Create a Python script that stores at least **4 pieces of information** about a character using variable boxes. Then, use those boxes to display a "Stat Board" on the screen.

### 🛠️ Step 1: Create your Variables
Pick a character (example: Mario, Elsa, Spider-Man). Create boxes for:
- `character_name`
- `power_level`
- `lives_left`
- `friend_count`

### 🛠️ Step 2: Display the Stats
Use the `print()` command to show the stats. 

---

### 🌟 Bonus Challenge (The AI Partner)
Use **Antigravity Gemini** to help you "Level Up" your character. Ask the AI:

> "I have a character named [Your Character] with [X] strength. Can you write a Python line that adds 50 more points to my `strength` variable?"

**Final Goal:** When you run your code, it should look like this:
```text
Character Name: Elsa
Power Level: 99
Lives Left: 3
Friend Count: 10
--- LEVEL UP! ---
Power Level: 149
```

---

### 🛡️ The Behind-the-Scenes Secret
By completing this challenge, you have just simulated how **Instagram** stores your profile info and how **Minecraft** stores your inventory. You are already thinking like a developer!
