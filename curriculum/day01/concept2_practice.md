# 🛠️ Concept 2 Practice: The Variable Vault

### 🚀 The 20-Problem Challenge Menu
*Teachers: Encourage students to experiment with different variable names—make them silly!*

#### 🟢 Group A: The Box Builders (Warmups)
1.  **The Score:** Create a variable named `score` and set it to 10.
2.  **The Namer:** Create a variable for your name and print it.
3.  **The Counter:** Create a variable `bottles` and set it to 99.
4.  **The Logic Switch:** Create a variable `is_happy` and set it to `True`.
5.  **The Color:** Create a variable `fav_color` and store a string inside.

#### 🟡 Group B: The Updaters (Changing Data)
6.  **The Level Up:** Start `level` at 1, then change it to 2 on the next line.
7.  **The Shopping Trip:** Start `wallet` at 20. Change it to 10 (you bought snacks!).
8.  **The Birthday:** Create an `age` variable. Update it to be one year older.
9.  **The Swap:** Create `apple = 5`. On the next line, change `apple` to 10.
10. **The Reset:** Create a variable `timer = 60`. Reset it to 0.

#### 🔴 Group C: The Bug Squasher (Fix the Boxes)
11. **Backwards Box:** `100 = money` (Fix the order!).
12. **Space Jam:** `my score = 500` (Fix the name—no spaces allowed!).
13. **Quote Quest:** `animal = Cat` (Fix the missing quotes for the string!).
14. **Case Confusion:** `lives = 3` then `print(Lives)` (Fix the capitalization!).
15. **Starting Number:** `1st_place = "Alex"` (Fix the name—cannot start with a number!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Inventory Guru:** Ask Gemini: "Give me 5 Python variable names I would need for a 'Spider-man' game and their starting values."
17. **The Data Scientist:** Ask Gemini: "What happens if I try to add a Number variable and a String variable together in Python?"
18. **The Naming Pro:** Ask Gemini: "What is 'snake_case' and why do Python coders use it for variable names?"
19. **The Real-World Modeler:** Ask Gemini: "Write me 3 variables that an 'Uber' app uses to track a ride."
20. **The Challenge:** Ask Gemini: "Give me a hard variable-naming puzzle for a beginner coder!"

### 🌟 Stretch Goal for "Flash" Students
Can you create a "Bank Account" script? 
1. Create a variable `balance`. 
2. Use `print` to show the balance.
3. Update the `balance` after "finding $50".
4. Update the `balance` after "spending $20".
*Goal: Show the history of the account on the screen!*
