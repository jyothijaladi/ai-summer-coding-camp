# 🛠️ Concept 2 Practice: Storing Secrets

### 📦 Building your First Box
Type this into your Python editor:

```python
score = 100
health = 3
age = 12

print(score)
print(health)
print(age)
```

### 🧪 Experiment Time: The Game Update
Now, let's try to update your variables. Try this:

```python
coins = 5
print(coins)

# You found a treasure chest!
coins = 50
print(coins)
```
Did the number change on the screen? That's the power of variables!

### 💡 Hint
In Python, we use the `=` sign to put something **INSIDE** the box. The name of the box always comes first!
`box_name = 10` (Correct ✅)
`10 = box_name` (Wrong ❌)
