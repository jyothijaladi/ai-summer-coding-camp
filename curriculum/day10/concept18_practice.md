# 🛠️ Concept 18 Practice: The Multi-Level Loop

### 🚀 The 20-Problem Challenge Menu
*Note: This is the most complex logic we've done yet! Take it slow.*

#### 🟢 Group A: Simple Grids (Warmups)
1.  **The Square:** Print a 3x3 square of asterisks (`*`).
2.  **Number Square:** Print a 2x2 grid where every number is the same.
3.  **The Row Label:** Print 3 rows, and inside each row print "A", "B", "C".
4.  **The Fast Count:** For each number 1-3, print numbers 10, 20, 30.
5.  **Multi-Print:** Print your name 3 times, and after each time, print "---" 3 times.

#### 🟡 Group B: Pattern Building
6.  **The Increasing Line:** In your first loop, print 1 star. In the second, print 2 stars.
7.  **The Coordinates:** Print `(row, col)` for a 3x3 grid (Example: `(0,0), (0,1)...`).
8.  **The Times Table:** Print the "2 Times Table" inside the "5 Times Table".
9.  **The Emoji Wall:** Print a 5x5 wall of mixed emojis.
10. **The Block Builder:** For each "Building", print 3 "Floors".

#### 🔴 Group C: The Bug Squasher (Fix the Stairs)
11. **Flat Logic:** Nested loop with no indent on the second loop.
12. **Variable Clashing:** Both loops using the same variable name `i` (Fix it to `i` and `j`!).
13. **The Infinite Slide:** A nested `while` loop inside a `for` loop that never stops.
14. **The Gatekeeper:** An `if` statement that blocks the inner loop from running.
15. **The Off-by-One:** A 5x5 grid that only prints 4x4 because of a `range` error.

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Pattern Pro:** Ask Gemini: "Write a nested loop that draws a 'Pyramid' of stars."
17. **The Grid Master:** Ask Gemini: "How do I use nested loops to create a 2D list (a list inside a list) for a Tic-Tac-Toe board?"
18. **The Clock Sim:** Ask Gemini: "Write a nested loop that prints every minute for every hour in a 12-hour day."
19. **The Real-World Modeler:** Ask Gemini: "How does a game like Minecraft use nested loops to put blocks in a 3D space?"
20. **The Challenge:** Ask Gemini: "Give me a hard nested loop challenge that involves the 'Modulo' operator to create a checkerboard pattern!"

### 🌟 Stretch Goal for "Flash" Students
Can you build a **"Calendar Printer"**? 
1. Create a list of 4 "Month" names.
2. For every month, print "Week 1", "Week 2", "Week 3", "Week 4".
3. **Extra Credit:** Inside every week, print "Day 1" through "Day 7"!
