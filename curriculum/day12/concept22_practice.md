# 🛠️ Concept 22 Practice: Functional Thinking (Reusability)

### 🚀 The 20-Problem Challenge Menu
*Note: Today's goal is to make your code "Short & Sweet"!*

#### 🟢 Group A: The DRY Principles (Warmups)
1.  **The Repeater:** Call a `heart()` function 10 times using a `for` loop.
2.  **The Double Stack:** Define two functions and call the second one inside the first.
3.  **The Updater:** Create a function that adds 10 to a global variable `total`.
4.  **The Multi-Sound:** Call `bark()` and `meow()` in a single function called `animal_sounds()`.
5.  **The Line Maker:** Write a function that prints a line of 20 dashes `---`.

#### 🟡 Group B: The Automation Bot
6.  **The Forest:** Use a loop to call a `draw_tree()` function 5 times.
7.  **The Party:** Use a loop and a list of names to call a `greet(name)` function for each person.
8.  **The Menu Bot:** Define `show_menu()`. Call it every time the user enters a loop.
9.  **The Health Bar:** Define `show_stats()`. Call it after every "Attack".
10. **The Pattern:** Use a loop to call a function that prints a random number.

#### 🔴 Group C: The Bug Squasher (Fix the Redundancy)
11. **The Long Way:** A script that prints "Hello" 5 times without a loop or function. (Convert it!).
12. **The Double Name:** `def hi(): print("Hi")` and `def hi(): print("Hello")` (Fix the name clash!).
13. **The Variable Ghost:** A function trying to use a variable that only exists in another function.
14. **The Missing Call:** An "Automation" script that defines 10 functions but never calls them.
15. **The Spelling Error:** Calling `spawn_zombie()` when the function is named `spawn_enemy()`.

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Code Cleaner:** Ask Gemini: "I have 20 lines of code that all do the same thing. How do I put them into a function to save space?"
17. **The Game Designer:** Ask Gemini: "Write three functions: `start_game()`, `play_round()`, and `end_game()`."
18. **The Logic Pro:** Ask Gemini: "What is 'DRY' in programming and why is it important?"
19. **The Real-World Modeler:** Ask Gemini: "Write a function for an ATM that checks if a user has enough money before letting them withdraw."
20. **The Challenge:** Ask Gemini: "Give me a challenge where I have to build a 'Robot' using 5 different functions (Legs, Arms, Head, etc.)."

---

### 🌟 Stretch Goal for "Flash" Students
Can you build a **"Game Engine"**? 
1. Define a function for `level_1()`, `level_2()`, and `level_3()`.
2. Inside each level, use `input()` to ask a question.
3. **If** they get it right, "Call" the next level function!
4. This is called **Chaining Functions**, and it's how big stories are built!
