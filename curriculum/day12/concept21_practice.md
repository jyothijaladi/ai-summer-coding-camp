# 🛠️ Concept 21 Practice: The Magic Machine (Functions)

### 🚀 The 20-Problem Challenge Menu
*Note: This is how we organize our code so it doesn't get "Messy"!*

#### 🟢 Group A: The First Creators (Warmups)
1.  **Simple Greet:** Define a function `greet()` that prints "Hi!". Call it once.
2.  **Triple Greet:** Call your `greet()` function 3 times in a row.
3.  **The Artist:** Define a function `draw()` that prints a smiley face `:)`.
4.  **The Sound Bot:** Define a function `meow()` that prints "Meow!".
5.  **The Weather:** Define a function `weather()` that prints "It's sunny out!".

#### 🟡 Group B: The Automation Bot
6.  **The Countdown:** Define a function that prints a `for` loop counting down from 5.
7.  **The Inventory Recall:** Define a function that prints out every item in your `backpack` list.
8.  **The Score Label:** Define a function that prints `f"SCORE: {score}"`.
9.  **The Daily Routine:** Define a function that prints: "Slept", "Ate", "Coded".
10. **The Game Over:** Define a function `game_over()` that prints a giant "RIP" and resets the score.

#### 🔴 Group C: The Bug Squasher (Fix the Machines)
11. **The Ghost Machine:** `def hi() print("Hi")` (Fix the missing colon!).
12. **The Order Error:** Calling a function before you define it at the top of the file.
13. **The Parentheses Trap:** `greet` (Fix it—you need the parentheses to call it!).
14. **The Indent Issue:** 
    ```python
    def my_func():
    print("Test") # Fix the alignment!
    ```
15. **The Spelling Error:** `deff hello():` (Fix the keyword!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Action Guru:** Ask Gemini: "I am building a 'Super Jump' move. Write a Python function for me called `super_jump()` that prints three action messages."
17. **The Library Pro:** Ask Gemini: "What is the difference between a 'Function' and a 'Method' in Python?"
18. **The Built-in Check:** Ask Gemini: "List 5 functions that are ALREADY built into Python (like `print()`)."
19. **The Real-World Modeler:** Ask Gemini: "Write a function used in a 'Smart Microwave' that prints 'Beep! Food is ready'."
20. **The Challenge:** Ask Gemini: "Give me a function puzzle that calls a SECOND function inside of it!"

### 🌟 Stretch Goal for "Flash" Students
Can you build a **"Command Center"**? 
1. Define 3 different functions: `attack()`, `defend()`, and `run()`.
2. Use a `while True` loop to ask the user "What do you do?".
3. **If** the user types "Attack," call the `attack()` function!
4. **If** they type "Quit," break the loop!
