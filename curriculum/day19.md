# 🕵️ Day 19: Project - The AI Detective Agency

**Time:** 9:00 AM - 12:00 PM
**Daily Goal:** Use AI to generate mysterious clues and save them into a "Case File" that you can read later to solve the crime.

---

## 🕒 9:00 AM – 10:40 AM: 🛠️ Guided Build – The Clue Generator 🔍
*Combining AI imagination with file storage.*

1.  **The Mystery Setup:** 
    - Use `input()` to choose a crime (e.g., "The Stolen Cookie"). 
2.  **AI Clues:**
    - Use a prompt to ask Gemini: "I am a detective investigating [crime]. Give me a mysterious clue that involves a footprint and a half-eaten snack."
3.  **Writing the Case File:**
    - Save the AI's response into a file named `case_file.txt`.
4.  **The AI Connection:**
    - Ask Gemini: "I want to save my AI responses but I don't want to lose the old clues. How do I use the 'a' (append) mode to add new clues to the bottom of my file?"

---

## 🕒 10:40 AM – 11:00 AM: 🥤 Mini-Break / Buffer

---

## 🕒 11:00 AM – 12:00 PM: 🏝️ The Sandbox Exploration
*Goal: Build the "Final Deduction" System.*

**Challenges for the Students:**
1.  **The Evidence Log:** Create a loop where the detective can "Search the Room." Each time, the AI gives a new clue and it gets saved to the file.
2.  **The Case Closer:** Write a function that reads the entire `case_file.txt` and asks the user: "Who do you think did it?" Save their guess to the file too.
3.  **The AI Boss Fight:** 
    - Open Antigravity Gemini.
    - Prompt: "I have a file with 5 clues. How do I write a Python script that counts how many lines are in the file and prints: 'You have found [X] clues so far'?"

### 🏆 Milestone:
*Students who finish can print out their "Official Case Report" — a full transcript of their investigation saved in their text file!*
