# 🛠️ Concept 5 Practice: The Word Wizard

### 🚀 The 20-Problem Challenge Menu
*Note: Quotation marks are your best friends today!*

#### 🟢 Group A: The String Construction Warmups
1.  **First Word:** Store your favorite food in a variable and print it.
2.  **Combine Strings:** Create `first_name` and `last_name`. Print them together.
3.  **The Shout:** Create a variable with a message and print it in ALL CAPS using `.upper()`.
4.  **The Whisper:** Print a message in all lower-case using `.lower()`.
5.  **Triple Message:** Print a string three times in a row using `* 3` (Example: `"Ha" * 3`).

#### 🟡 Group B: The String Detective (Indices & Methods)
6.  **First Letter:** Print the first character of the string `"Python"` (Slot 0).
7.  **Last Letter:** Print the last character of your name.
8.  **The Length:** Use `len()` to print how many letters are in your full name.
9.  **Middle Section:** Slice the string `"Programming"` to only show `"Gram"`.
10. **Find the Letter:** Use `.find()` to see if the letter "z" is in your favorite fruit.

#### 🔴 Group C: The Bug Squasher (Fix the Strings)
11. **Missing Wrappers:** `name = Alex` (Fix the missing quotes!).
12. **Mismatched Shoes:** `greeting = "Hi'` (Fix the mixing of double and single quotes!).
13. **Forbidden Math:** `"10" + 5` (Fix the code—turn the string into an integer or vice-versa!).
14. **Variable Confusion:** `print("score")` (When the goal was to print the value inside the `score` box!).
15. **Gap Fixer:** `print(Hello World)` (Fix the missing quotes and space!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Storyteller:** Ask Gemini: "Write a 3-sentence story and store each sentence in a Python variable named `s1`, `s2`, and `s3`."
17. **The Emoji Artist:** Ask Gemini: "How do I print a string that contains a bunch of sparkling star emojis?"
18. **The Formatter:** Ask Gemini: "What is the difference between `.capitalize()` and `.title()` in Python strings?"
19. **The Security Guard:** Ask Gemini: "How do I check if a string contains a special character like '@' using Python?"
20. **The Challenge:** Ask Gemini: "Give me a hard string-slicing puzzle for a beginner coder!"

### 🌟 Stretch Goal for "Flash" Students
Can you build a **"Name Scrambler"**?
1. Ask the user for their name.
2. Print it backwards.
3. Print it 100 times in a row.
4. Print only the first and last letters.
5. Print the name in alternating Case (like "PhYtHoN")!
