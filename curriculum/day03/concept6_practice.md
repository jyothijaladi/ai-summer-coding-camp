# 🛠️ Concept 6 Practice: The f-String Machine

### 🚀 The 20-Problem Challenge Menu
*Teachers: This is where kids start to feel like "Real" app developers!*

#### 🟢 Group A: The f-String Basics (Warmups)
1.  **Simple Swap:** Create a variable `name`. Print `f"Hello, {name}!"`.
2.  **Age Statement:** Create `age = 12`. Print `f"I am {age} years old."`.
3.  **The Math Lab:** Print an f-string that calculates `10 * 10` inside the braces.
4.  **Color Splash:** Create `color = "Green"`. Print `f"The grass is {color}."`.
5.  **Multi-Power:** Create `hero` and `power` variables. Print them in one f-string.

#### 🟡 Group B: The App Developer (Simulating Apps)
6.  **The Uber Bot:** Create `driver` and `minutes`. Print `f"Your driver {driver} is {minutes} mins away."`.
7.  **The Bank Statement:** Create `balance = 100`. Print `f"Your current balance is ${balance}."`.
8.  **The Achievement:** Create `player` and `score`. Print `f"🏆 Congrats {player}! New High Score: {score}"`.
9.  **The Chef:** Create `ing1` and `ing2`. Print `f"Today we are cooking {ing1} with a side of {ing2}."`.
10. **The Weatherbot:** Create `temp`. Print `f"Brrr! It's {temp} degrees outside. Wear a coat!"`.

#### 🔴 Group C: The Bug Squasher (Fix the Braces)
11. **The Forgotten 'f':** `"Total is {10+5}"` (Fix it so it does the math!).
12. **Wrong Brackets:** `f"Hi (name)"` (Fix the brackets!).
13. **Missing Variable:** `f"I love {food}"` (Fix it by creating the `food` variable first!).
14. **Extra Spaces:** `f" { name } "` (Clean up the spaces if they look weird!).
15. **Quote Mismatch:** `f"He said {"Hello"}"` (Fix the inner quotes—HINT: use single quotes inside double quotes!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Story Generator:** Ask Gemini: "Give me an f-string template for a 'Star Wars' opening crawl where I can swap the planet and the hero name."
17. **The Currency Pro:** Ask Gemini: "How do I use an f-string to show a number as a dollar amount with two decimal places (Example: $10.00)?"
18. **The Date Master:** Ask Gemini: "How do I format the current date inside an f-string in Python?"
19. **The Logic Mixer:** Ask Gemini: "Can I use an `if` statement inside an f-string? Show me a small example."
20. **The Challenge:** Ask Gemini: "Give me a complex 'Mad Libs' f-string template with at least 5 different variables for me to fill in!"

---

### 🌟 Stretch Goal for "Flash" Students
Can you build a **"Player Stat Card"** using one single f-string?
Use `\n` (the newline character) inside your f-string to print multiple lines at once!
Example: `f"Name: {n}\nHP: {h}"`
