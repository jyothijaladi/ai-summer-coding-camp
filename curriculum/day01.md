# 👋 Day 1: Hello World & The Magic Boxes

**Time:** 9:00 AM - 12:00 PM
**Daily Goal:** Make the computer speak and learn how to store secrets in "Variable Boxes."

---

## 🕒 9:00 AM – 9:40 AM: Concept 1 – The Hello World "Magic" ✨
*Every great programmer starts with two words.*

1.  **The Print Command:** 
    - Explain: `print()` is a megaphone. Anything you put inside the parentheses gets shouted to the screen.
    - Activity: Everyone types: `print("Hello World!")`
2.  **The "Magic Message" (Strings Preview):**
    - Explain that the quotes `""` are like a package. We are telling Python: "Don't execute this, just show it!"
3.  **The AI Connection:**
    - Ask Gemini: "Why do programmers always start with 'Hello World'?"

---

## 🕒 9:40 AM – 10:00 AM: 🍎 Brain Break (20 mins)
*Get up, stretch, and step away from the screen!*

---

## 🕒 10:00 AM – 10:40 AM: Concept 2 – Variable "Boxes" 📦
*Storing numbers for later.*

1.  **The Box Analogy:**
    - A variable is a box with a label on the front. 
    - Inside the box, we put a value (like a number).
2.  **Number Boxes:**
    - Let's create our first boxes:
      ```python
      score = 0
      lives = 3
      age = 10
      ```
3.  **The AI Connection:**
    - Task: Ask Gemini: "Give me an example of how a video game uses variable boxes to keep track of a player."

---

## 🕒 10:40 AM – 11:00 AM: 🥤 Mini-Break / Buffer
*Quick water break or finishing up the previous concept.*

---

## 🕒 11:00 AM – 12:00 PM: 🏝️ The Sandbox Exploration (Level 1)
*Goal: Create a "Video Game Stats" display.*

**Challenges for the Students:**
1.  **Level 1 (The Courier):** Create a variable named `coins` and set it to 100. Then use `print(coins)` to show it on the screen.
2.  **Level 2 (The Manager):** Create three boxes: `strength`, `speed`, and `luck`. Assign them numbers between 1 and 10.
3.  **The AI Boss Fight:** 
    - Open Antigravity Gemini.
    - Prompt: "I have a variable box named `health` set to 50. Write a code snippet that prints 'Your current health is:' and then prints the `health` variable on the next line."
    - Copy the code and try to run it!

---

### 💡 Teacher's Tip:
*If a student finishes early, ask them to change their `health` variable from 50 to 100 and run the code again. This helps them see that the `print` command stays the same, even when the box's contents change!*
