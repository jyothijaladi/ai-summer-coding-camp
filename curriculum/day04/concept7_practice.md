# 🛠️ Concept 7 Practice: Listening to the Player

### 🚀 The 20-Problem Challenge Menu
*Teachers: This is the first time students will feel like the computer is actually "talking" to them!*

#### 🟢 Group A: The First Conversations (Warmups)
1.  **The Greeting:** Ask the user for their name and print "Hello [Name]!".
2.  **Color Quest:** Ask for their favorite color and print "I love [Color] too!".
3.  **The Echo:** Ask for a word and print it 5 times in a row.
4.  **Action Request:** Ask "What do you want to do?" and print "Going to [Action] now...".
5.  **Shout Back:** Ask for a message and print it in ALL CAPS.

#### 🟡 Group B: The Math & Input Mix (The Integer Wrap)
6.  **The Age Calculator:** Ask for their age and print how old they will be in 1 year. (*Hint: Use `int()`!*)
7.  **The Pizza Slice:** Ask how many pizzas they have and multiply it by 8 slices.
8.  **Double Trouble:** Ask for a number and print that number multiplied by 2.
9.  **The Allowance:** Ask how much they get per week and print how much they get in a year (52 weeks).
10. **The Splitter:** Ask for a total bill amount and divide it by 2 friends.

#### 🔴 Group C: The Bug Squasher (Fix the Listener)
11. **The Ghost Cursor:** `input("Enter name")` (Fix it by saving it to a variable!).
12. **The Crasher:** `age = input("Age?")` followed by `print(age + 1)` (Fix the math error!).
13. **The Promptless:** `x = input()` (Fix it by adding a friendly message inside the parens!).
14. **Variable Mixup:** `name = input("Hi")` followed by `print(Name)` (Fix the capital letter!).
15. **The Unfinished:** `input("Enter code"` (Fix the missing parenthesis!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Interrogator:** Ask Gemini: "Help me write 3 interesting questions an AI would ask a human."
17. **The Input Transformer:** Ask Gemini: "How do I take a user input like 'red' and change it into 'RED' immediately in one line of code?"
18. **The Validation Pro:** Ask Gemini: "How do I check if the user only typed numbers and not letters?"
19. **The Real-World Modeler:** Ask Gemini: "Write a small script that asks for a username and a password (using input) and prints 'Profile Created'."
20. **The Challenge:** Ask Gemini: "Give me a puzzle where I have to take three different inputs and combine them into a secret sentence!"

### 🌟 Stretch Goal for "Flash" Students
Build a **"Fortune Teller"** program! 
1. Ask the user for their name.
2. Ask for a lucky number.
3. Ask for a favorite color.
4. Use f-strings to "Predict" their future based on those pieces of data!
