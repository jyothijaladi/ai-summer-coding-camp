# 🛠️ Concept 8 Practice: The Logic Gate (If/Else)

### 🚀 The 20-Problem Challenge Menu
*Note: Don't forget the colon `:` and the indent (4 spaces)!*

#### 🟢 Group A: Simple Decisions (Warmups)
1.  **The Secret Number:** If a variable `num` is 7, print "Lucky!".
2.  **Weather Check:** If `is_raining` is True, print "Take an umbrella". Else, print "Enjoy the sun".
3.  **Positive Thinking:** Check if a number is greater than 0.
4.  **Equal Match:** Check if two strings `pass1` and `pass2` are the same.
5.  **Age Gate:** Check if `age` is 13 or older.

#### 🟡 Group B: The Game Logic Simulator
6.  **The Winner:** If `score` is 100 or more, print "WINNER". Else, print "KEEP TRYING".
7.  **Empty Bag:** If `items_in_bag` is 0, print "Find some loot!".
8.  **Boss Room:** If `player_has_key` is True, print "Enter Room". Else, print "Locked!".
9.  **Speed Trap:** If `speed` is over 60, print "Slow down!".
10. **Game Over:** If `lives` is less than 1, print "GAME OVER".

#### 🔴 Group C: The Bug Squasher (Fix the Gates)
11. **Missing Colon:** `if x == 10 print(x)` (Fix it!).
12. **The Single Equal:** `if x = 10:` (Fix the assignment vs comparison error!).
13. **The Indent Issue:** 
    ```python
    if x > 5:
    print("Higher!") # Fix the alignment!
    ```
14. **The Else Trap:** `else print("Low")` (Fix the missing colon!).
15. **Case Mixup:** `if name == "Alex":` when the user typed `alex` (Fix the comparison!).

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Movie Critic:** Ask Gemini: "Write a Python script that asks for a movie rating and prints 'Awesome' if it's over 8 and 'Meh' if it's not."
17. **The Emoji Guard:** Ask Gemini: "How do I write an `if` statement that checks if a string is exactly one emoji?"
18. **The Password Pro:** Ask Gemini: "Write an if/else system that checks both a username and a password."
19. **The Logic Mixer:** Ask Gemini: "What is `elif` and how is it different from a regular `else`?"
20. **The Challenge:** Ask Gemini: "Give me a hard if/else puzzle involving three different numbers!"
