# 🛠️ Concept 27 Practice: The Crash Shield (Error Handling)

### 🚀 The 20-Problem Challenge Menu
*Note: Today we make our programs "Un-crashable"!*

#### 🟢 Group A: The Protection Warmups
1.  **Simple Try:** Put `print("Ok")` inside a `try` block.
2.  **Force a Mistake:** Try to divide 10 by 0. Catch the error and print "No zero math!".
3.  **Input Guard:** Ask for a number. If they type a word, catch it and say "Invalid Input".
4.  **The Mystery Key:** Look for a key in a dictionary that doesn't exist. Catch the error.
5.  **List Leap:** Try to print the 5th item of a list with only 3 items. Catch the error.

#### 🟡 Group B: The Secure App Logic
6.  **The Bank Vault:** Ask for a PIN (number). Catch the error if they type letters.
7.  **The Color Mixer:** Take two inputs. If either is missing, print "Color Error".
8.  **The Health Bar:** Try to subtract "Full" from `hp` (math error). Catch it!
9.  **The Inventory Search:** Use `input` to find an item in a list. If not found, catch the "ValueError".
10. **The Multi-Shield:** Put 3 different types of risky code inside ONE `try` block.

#### 🔴 Group C: The Bug Squasher (Fix the Shield)
11. **The Ghost Except:** `try: print("Hi")` (Fix it—every `try` needs an `except`!).
12. **Wrong Keyword:** `catch: print("Error")` (Fix it—Python uses `except`!).
13. **Indentation Gate:** `try:` with the code below it not pushed to the right.
14. **The Colon Trap:** `except print("Oops")` (Fix the missing colon!).
15. **The Unreachable Shield:** A `try` block that has no code inside it.

#### 💎 Group D: The AI Whisperer (Gemini Missions)
16. **The Logic Guard:** Ask Gemini: "How do I catch TWO different types of errors in one single `try` block?"
17. **The Message Catcher:** Ask Gemini: "How do I capture the EXACT error message the computer gives and print it (using `as e`)?"
18. **The Final Result:** Ask Gemini: "What does the `finally:` block do in error handling?"
19. **The Real-World Modeler:** Ask Gemini: "Write a script that tries to open a website and prints 'Internet Error' if it fails."
20. **The Challenge:** Ask Gemini: "Give me a challenge where I have to use `try/except` inside a `while` loop so the user has to keep trying until they get the math right!"

### 🌟 Stretch Goal for "Flash" Students
Can you build a **"Super-Secure Calculator"**? 
1. Ask the user for `num1`, `num2`, and an `operator`.
2. Put the math inside a `try` block.
3. Catch `ValueError` (if they type letters).
4. Catch `ZeroDivisionError` (if they divide by 0).
5. Catch all other errors with an "Unexpected Error" message.
6. Make it run forever in a `while` loop until they type "Quit"!
