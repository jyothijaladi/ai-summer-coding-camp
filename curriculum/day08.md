# 🔁 Day 8: The "For" Loop (Automation)

**Time:** 9:00 AM - 12:00 PM
**Daily Goal:** Learn how to use Loops to repeat a task for every item in a collection.

---

## 🕒 9:00 AM – 9:40 AM: Concept 1 – Looping through Lists 🔁
*How to avoid typing the same thing 100 times.*

1.  **The `for` Loop:** 
    - `for item in list: print(item)`
    - Explain that Python picks up each item from the list, one by one, and names it `item` (or any name you choose) temporarily.
2.  **The "Indentation" reminder:**
    - Just like `if` statements, the code inside the loop must be indented (4 spaces).
3.  **The AI Connection:**
    - Ask Gemini: "Show me a for loop that takes a list of numbers and prints each number multiplied by 2."

## 🕒 10:00 AM – 10:40 AM: Concept 2 – The Counting Rocket (`range`) 🚀
*Creating loops that run a specific number of times.*

1.  **The `range()` function:**
    - `for i in range(5): print(i)`
    - Explain that `range(5)` creates a sequence from 0 to 4.
2.  **Counting backwards / Jumping:**
    - `range(10, 0, -1)` counts down from 10 to 1.
3.  **The AI Connection:**
    - Task: Ask Gemini: "How do I use `range` to print all the even numbers from 0 to 20?"

### 💡 Teacher's Tip:
*Students often get confused about the variable name in `for item in list:`. Tell them: "You are naming the single item that the loop is holding right now. It can be called `x`, `bird`, or `pizza`, but it's best to call it something that makes sense!"*
