# 🎒 Teacher's Setup & Prerequisites Guide (Grades 3-8)

Welcome, Teacher! Before your first day of camp, you need to make sure both the classroom and the kids are ready. Since these kids are in 3rd to 8th grade and have **zero** prior knowledge of coding or AI, everything must be explained like magic being turned into science.

## 1. What the Kids Need (Physical Setup)
Each student must have:
*   **A Laptop or Desktop Computer:** Tablets/iPads are incredibly difficult for typing python code. A computer with a real keyboard is a must!
*   **Internet Access:** Our AI tools live on the internet! 

## 2. The Software (What to Install)
Before Day 1, please ensure these three things are installed on every computer:

### A. Python
*   **What is it?** It's the engine that runs our code. Without it, the computer doesn't understand Python language.
*   **How:** Download from python.org and install it. (Make sure to check the box that says "Add Python to PATH" during installation!)

### B. Visual Studio Code (VS Code)
*   **What is it?** This is our "Workshop". It’s just a super fancy text editor where we type our code. It highlights words in different colors to make reading easier.
*   **How:** Download from code.visualstudio.com. 

### C. Antigravity Gemini (The AI LLM)
*   **What is it?** This is our "Robot Assistant". Gemini is a Large Language Model (LLM). Imagine reading every book in every library on Earth, and then being able to answer questions based on those books. That is an LLM! 
*   **Why Antigravity?** It's specifically designed to help us code and understand agentic workflows safely. 

---

## 3. How to Explain These Concepts to 8-13 Year Olds

When kids sit down on Day 1, they will be staring at scary black screens with colorful text. You need to demystify it instantly! Use these analogies:

### What is a Computer Program?
> *"A computer is actually very dumb! It is like a super-fast chef who is completely blind and deaf. If you want the chef to bake a cake, you can't just say 'Bake a cake'. You have to write: '1. Open fridge. 2. Grab eggs. 3. Crack eggs into bowl.' A computer program (Code) is just a very specific recipe you write for the computer to follow."*

### What is Visual Studio Code?
> *"Imagine an artist. An artist could paint on a dirty napkin, but they prefer a beautiful art studio with nice lighting and easels. Visual Studio Code is our Art Studio. We *could* write code in Microsoft Word, but VS Code is a special room built just for coders. It colors our text (variables are blue, strings are green) so our eyes don't get tired!"*

### What is AI / an LLM?
> *"Raise your hand if you've ever used Auto-Correct on your phone. When you type 'I love p...', your phone guesses 'pizza'. That's a tiny AI guessing the next letter. A Large Language Model (LLM) like Gemini is a giant brain that read the whole internet. When we ask it a question, it is mathematically guessing the best possible next word to help us. It's not alive, it's just really, really good math!"*

### What is "Agentic Programming"?
> *"Normally, you are the boss, and the computer is the worker. You tell it to do math, it does math. In 'Agentic Programming', we make the AI the boss! We give the AI a robot body (tools), and we let the AI decide when to use a calculator, when to search the internet, and when to talk to humans. You are building JARVIS from Iron Man!"*

## 4. The First 15 Minutes (Icebreaker)
Do not open the computers right away!
1. Ask: "Who plays video games?" (Everyone will raise their hand).
2. Ask: "How do you think Mario knows to jump when you press the A button?"
3. Explain: "Someone had to write a recipe that said: `if button == A: jump()`. By the end of this camp, YOU will know how to write those recipes!"
