# 🚀 30-Day Python & Agentic AI Summer Camp Curriculum 🤖

Welcome to the ultimate summer camp guide for teaching kids Python and Agentic Programming using Antigravity Gemini! This curriculum is designed for 30 days (6 weeks, Monday-Friday), with each day taking ~1 to 1.5 hours.

**Goal:** Teach Python basics, empower kids to build agent-like applications, and encourage them to innovate by building their own projects.
**Key Focus:** Fun, interactive, and heavy on "Agentic AI" (using AI to generate code, write logic, and act as a smart assistant within their projects).

---

## 📅 Week 1: Introduction to Python & AI Assistants (The Basics)
*Goal: Understand the absolute basics of Python and how to prompt an AI (Gemini) correctly to help us write code.*
- **Day 1:** 🐍 Hello World & Meeting Gemini (Variables, `print`, AI prompt basics).
- **Day 2:** 🔢 Data Types & Asking the Right Questions (Strings, Integers, and how AI understands text vs numbers).
- **Day 3:** ⚖️ Decisions, Decisions... (If/Else statements, Logic).
- **Day 4:** 🔁 The Loop-de-Loop (For & While loops, asking Gemini to generate giant patterns for us!).
- **Day 5:** 🛠️ **Project 1:** Interactive Story Maker! (Using Python + Gemini to write Mad Libs-style interactive stories).

## 📅 Week 2: Functions & Designing Tools for AI
*Goal: Learn how to group code into reusable blocks, and conceptually understand how an AI "uses tools".*
- **Day 6:** 📦 Functions - Putting code in a box (Defining functions, parameters).
- **Day 7:** ↩️ Return Values - Getting answers back from our boxes.
- **Day 8:** 🧠 Agentic AI Concepts - What is a "Tool" for an AI? (Writing simple Python functions that an AI *could* theoretically use).
- **Day 9:** 📋 Lists - Keeping track of our cool stuff.
- **Day 10:** 🛠️ **Project 2:** Smart Quiz Game (Using lists and functions, with Gemini helping generate the trivia questions!).

## 📅 Week 3: Dictionaries & Connecting to the World
*Goal: Learn advanced data structures and connect Python to the real world (APIs).*
- **Day 11:** 📖 Dictionaries - Organizing data like a pro (Key-Value pairs).
- **Day 12:** 🌐 APIs - What are they? (Connecting to a simple public API like weather or space facts).
- **Day 13:** 📡 Fetching Data - Using the `requests` library in Python.
- **Day 14:** 🤖 AI Data Formatter - Letting Gemini take messy data and rewrite it nicely for humans.
- **Day 15:** 🛠️ **Project 3:** Space/Weather App with an AI Personality!

## 📅 Week 4: The Agentic Core (Building a Bot)
*Goal: Build an actual AI agent that remembers things and uses basic tools.*
- **Day 16:** 🤖 What is a Bot/Agent? (State, Memory, and Prompts).
- **Day 17:** 🎭 Designing our Bot's Personality (System Prompts).
- **Day 18:** 🔧 Giving the Bot a Python "Tool" (Making a calculator function that the bot recommends using).
- **Day 19:** 💬 Making the Chat Loop (Keeping the conversation going using a `while` loop).
- **Day 20:** 🛠️ **Project 4:** Console AI Assistant! (A bot that chats and runs basic python commands for you).

## 📅 Week 5: Making it Visual (Web Apps / UI)
*Goal: Move away from the black terminal and build real UI apps using a library like Streamlit or Gradio.*
- **Day 21:** 🖥️ Intro to Visual Apps (What is Streamlit/Gradio?)
- **Day 22:** 🔘 Buttons, Inputs, and Text on a screen.
- **Day 23:** 🔌 Connecting our Python AI Chatbot to the Web Interace.
- **Day 24:** 🎨 Customizing the Look (Adding images, emojis, and layouts).
- **Day 25:** 🛠️ **Project 5:** Web-based AI Chatbot App!

## 📅 Week 6: Innovation Showcase (Final Projects)
*Goal: Kids brainstorm, design, build, and present their own innovative Agentic AI application.*
- **Day 26:** 💡 Brainstorming & Planning - Using Gemini to expand on project ideas.
- **Day 27:** 🏗️ Building V1 - Getting the core logic and AI connection working.
- **Day 28:** 🎨 Building V2 - Adding the UI and making it look awesome.
- **Day 29:** 🐛 Bug Squashing & Polish - Learning how to ask AI to fix specific errors.
- **Day 30:** 🏆 **Final Showcase!** Presenting the apps to the class and parents!

---
## ✨ Important Guidelines for the Daily Documents
Each day's lesson will include:
1. **The Core Concept:** Explained in kid-friendly terms (with emojis/analogies).
2. **Kid Q&A "Why is this useful?":** Real-world applications to keep them engaged.
3. **The AI Connection:** How Gemini or Agentic Programming relates to this topic.
4. **Hands-on Coding:** 2-3 short, highly interactive coding steps.
5. **Challenge of the Day:** A small puzzle or stretch goal.
