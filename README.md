# 🤖 AI Summer Coding Camp: From Screen Consumer to AI Creator

Welcome to the **AI Summer Coding Camp** curriculum—a 4-week intensive program designed for kids aged **9 to 14**. 

This program moves beyond traditional "syntax-first" teaching. Instead, we use **Agentic AI** (powered by Antigravity Gemini) and **Dual-Track Analogies** to help students build real, persistent games and applications.

---

## 🚀 The Program Vision
Our goal is to empower students to stop just playing games and start building them. By partnering with AI, students can bridge the gap between their imagination and their technical skills.

### Key Features:
*   **Dual-Track Analogies:** Every concept is explained through both **Action (Gaming/Sports)** and **Creative (Arts/Lifestyle)** lenses.
*   **Agentic AI Focus:** Students learn to use AI as a collaborator, debugger, and narrative engine.
*   **Project-Based Learning:** A "Level Up Challenge" or "Hero Project" every single day.
*   **Persistence Mastery:** Week 4 focuses on File Handling, allowing students to create games that save progress.

---

## 📅 The 4-Week Roadmap

### Week 1: The Logic Architect 🧩
- **Focus:** Variables, Numbers, Strings, and If/Else logic.
- **Hero Project:** The AI Escape Room.

### Week 2: The Loop Master 🔁
- **Focus:** Lists, Tuples, and Automation (For/While loops).
- **Hero Project:** The Virtual AI Pet (Tamagotchi).

### Week 3: The System Designer ⚙️
- **Focus:** Dictionaries, Sets, and Functions.
- **Hero Project:** The Boss Battle RPG.

### Week 4: The Innovation Lab 🚀
- **Focus:** File Persistence (Reading/Writing) and Project Marathons.
- **Hero Project:** The Infinite Adventure Book.
